# Xfce Theme Empty

The most minimalistic window manager theme for Xfce because there are *no* window borders at all!

![A desktop screenshot of the 'empty' theme](./screenshot.png)

## Installation

```sh
# make install
```

![xfwm4-settings](./xfwm4-settings.png)

## Uninstallation

```sh
# make uninstall
```
